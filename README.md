# Talleres PICTOS.net

Sitio web para los talleres de co-diseno participativo de PICTOS.net.

Presentacion construida con Reveal.js que cubre: que es el diseno participativo, como se aplica al diseno de herramientas de comunicacion aumentativa con IA, objetivos y estructura de los talleres, y convocatoria a participar.

## Uso

Abrir `index.html` en un navegador. La presentacion carga Reveal.js y la tipografia Lexend desde CDN.
